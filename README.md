Пример использования открытых библиотек https://dlib.net/, https://opencv.org/ в быту.

Назначение:
Поиск изображений в домашнем каталоге фотографий, содержащих указанные лица

Использование
1. Построение или обновление индекса дескрипторов каталога исходных изображений
2. Построение или обновление индексов дескрипторов искомых лиц
3. Поиск, по предварительно построенным индексам, или напрямую по снимках (трудоёмкий вариант)

Примечание:
Это не образец кода для подражания - смотрите документацию по вышеуказанным библитекам и документацию по языку https://gcc.gnu.org/onlinedocs/
Это пример варианта использования "artificial neural network (ANN)" в домашней библиотеке фотографий.
